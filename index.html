Simple Webpack workflow environment
Features: SASS, ES6/ES7, autoprefixer, minifications, dev server sith Hot Module Replacement, working with imgs, file structure, simple prod build with multiple entries, and other...
Starting Steps:
  1. sudo npm install (if some errors: sudo npm install --unsafe-perm=true --allow-root)
  2. npm run start (for development)
  3. npm run build (for bundling prod)
